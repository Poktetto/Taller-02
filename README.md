Este Github contiene el Taller 02 de Programación Orientada a Objetos, en este Taller en especifico se realizo un juego de simulación de la famosa franquicia de Pokémon que permite las funciones básicas de un juego de Pokémon como capturar pokémones combatir contra lideres de gimnasio y alto mando, ordenar pokémones en el equipo y en el PC, entre otros. También este juego posee un sistema de guardado en base a txt.

El paquete de este juego es  "Taller02", su clase main es "App.java" y una clase a destacar es "Pokemones.java" esta siendo la clase encargada de la creación de pokémones, las otras clases son "Gimnasio.java", "TablaDeTipos.java" y "AltoMando.java"
